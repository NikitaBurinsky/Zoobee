# Zoobee.Infrastructure.Parsers

Модуль инфраструктуры, отвечающий за сбор (scraping), хранение и первичную обработку данных с внешних сайтов зоотоваров.

## Назначение
Система предназначена для автоматизированного скачивания веб-страниц, отслеживания изменений и подготовки "сырых" данных (Raw HTML/XML) для последующей бизнес-трансформации.

## Ключевые возможности
* **Управление очередью**: Умное планирование повторных попыток скачивания (`NextTryAt`).
* **Seeding (Посев)**: Автоматическая загрузка стартовых точек входа (Sitemaps, каталоги) через конфигурацию.
* **История изменений**: Хранение всех версий HTML-кода для каждого URL.
* **Типизация задач**: Разделение на Sitemaps, Листинги и Страницы товаров.

## Быстрый старт
1.  Убедитесь, что строка подключения к БД настроена корректно.
2.  Настройте начальные ссылки в `appsettings.json` (раздел `ScrapingSeeding`).
3.  При запуске приложения сработает `ScrapingSeeder`, который наполнит базу задачами.
